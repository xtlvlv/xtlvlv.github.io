使用swagger文档
  1. swagger是对api的一种汇总。
  2. 需要在 conf/app.conf 打开开关：EnableDocs = true
  3. 运行项目的时候指定：bee run -gendoc=true -downdoc=true
  4. 默认文档路径是：http://localhost:8080/swagger/
